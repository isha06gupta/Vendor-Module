# 📦 Vendor Module – CEIL

A web-based vendor registration and login system built using **Spring Boot**, **HTML**, **CSS**, and **JavaScript**. Includes OTP-based authentication via email.

---

## 🚀 Features

- 🔐 User registration with email verification
- 🔁 Login with OTP
- 📬 Email integration using JavaMail
- 🎨 Responsive UI with dropdowns and modals
- ✅ Google reCAPTCHA integration
- 🗃️ MySQL database for user storage

---

## 🛠️ Tech Stack

- **Backend:** Spring Boot, Java, Maven
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **Email Service:** JavaMailSender
